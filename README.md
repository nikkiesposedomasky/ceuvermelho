## Integrantes
- Nikki
- Yan ribeiro

## Descrição

Céu Vermelho é um jogo desenvolvido em Ren'Py de terror psicologico que se passa em uma cidade onde o céu foi cobrido pela cor vermelha 
e os personagens precisam descobrir o que aconteceu e onde está todo mundo.

## Gênero
Visual Novel

## Público-alvo

Jovens e adultos interessados em jogos narrativos de terror como: Doki doki Literature Club

## Tecnologias utilizadas

- Ren'Py
- Python
- GitHub

## Como executar

1. Baixe o arquivo do jogo (que provavelmente vai ser lançado no itch.io)
2. extraia o arquivo
3. abra a pasta e inicie o .exe
